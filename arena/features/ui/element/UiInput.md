# 元素-UiInput

![](/QQ20240918-150346.png)


这里的UiInput是输入框，可以让玩家输入文本信息，传递给服务器。


## UiInput 属性
此元素继承至 [UiText](./UiText)。
| 属性        |      解释      | 
| ------------- | ----------- | 
| placeholder      | 输入框的未输入时文本提示内容。 | 
| placeholderColor      |   输入框显示的占位文本的颜色。   |   
| placeholderOpacity |   输入框提示文本的不透明度。   |
| isFocus |  输入框是否聚焦。    |
