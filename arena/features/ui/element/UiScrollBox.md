# 元素-UiScrollBox

![](/QQ20240918-145736.png)


- 滚动框本身是一类特殊的框架（已有元素），享有框架有的所有属性
- 滚动框由滚动条及内容画布构成
    - 滚动条区域
        - 标识当前滚动框内容展示的内容在内容画布上的相对定位关系的元素，允许拖动调整相对定位
        - 滚动条仅在当前可滚动（内容画布尺寸大于滚动框时）且开启了滚动条展示时才显示
        - 滚动条将占据滚动框的部分面积，位于滚动框顶层（会遮蔽下方展示物）
        - 滚动交互方式
            - 滚动条
                - 拖动（可见）
                - 鼠标滚轮滚动
            - 画布区域
                - 拖拽移动
    - 内容画布
        - 滚动框内容可容纳的最大内容展示区域，定义了滚动框本身可滚动区域的大小
