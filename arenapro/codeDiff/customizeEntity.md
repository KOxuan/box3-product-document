# 自定义entity类与方法
::: warning
 ⚠在JS开发中，许多人倾向于直接在GameEntity类下自定义属性/方法，以适应不同玩家或模型的数据需求。尽管TS也支持这种做法，但不建议使用。

如果官方在未来的更新中添加了与用户自定义属性同名的属性或方法，这将导致命名冲突。这种冲突可能导致代码无法正常运行，甚至引发严重的bug。

:::
