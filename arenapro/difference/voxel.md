
# 数据空间
:::warning
该API文档：[GameVoxels](https://docs.box3lab.com/api/GameVoxels/operate.html)
:::

:::tip
**修改原因：**  
在原先的官方d.ts文件中，该API接口的方块数据被宽泛地定义为`number`或`string`类型，这可能会引发一个问题：允许输入任意数据，进而导致运行时因无法获取对应方块而产生BUG。

因为神岛方块是有限的，我们引入了自定义类型来严格规范输入数据。这一举措不仅确保了数据的合法性，还显著提升了代码的可读性和错误检查能力。
:::
